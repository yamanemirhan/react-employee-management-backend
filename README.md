# react-employee-management-backend

## Backend  
npm install  
npm run start  

## Don't forget to put your own MONGO_URI in config file.

## Backend Base URL http://localhost:4000/  

# Backend Routes  
## Add Employee  
/api/employees  
##  Get All Employees  
/api/employees  
##  Get Single Employee  
/api/employees/:id  
##  Delete Employee  
/api/employees/:id  
##  Update Employee  
/api/employees/:id
